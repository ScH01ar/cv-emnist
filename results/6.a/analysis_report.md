# EMNIST Balanced 四种模型性能对比报告

## 1. 实验目的

本实验对 `MLP`、`CNN`、`ResNet` 和 `ViT` 四种模型在 `EMNIST Balanced` 测试集上的表现进行统一评估，重点比较它们的分类准确率、精确率、召回率和 F1 分数，并结合模型结构分析其性能差异。

## 2. 指标汇总

| model_name | accuracy | precision | recall | f1     |
| ---------- | -------- | --------- | ------ | ------ |
| ResNet     | 0.9016   | 0.9021    | 0.9016 | 0.9014 |
| ViT        | 0.8930   | 0.8937    | 0.8930 | 0.8923 |
| CNN        | 0.8914   | 0.8920    | 0.8914 | 0.8908 |
| MLP        | 0.8865   | 0.8895    | 0.8865 | 0.8851 |

从测试结果来看，`ResNet` 的整体表现最好，测试集准确率达到 `0.9016`；`MLP` 的表现相对最弱，准确率为 `0.8865`。两者准确率差值为 `0.0152`。

四项指标的排序基本一致，说明四个模型之间的性能差异较稳定，并不是仅在某一个单独指标上出现偶然波动。

## 3. 参数规模对比

| model_name | parameters |
| ---------- | ---------- |
| MLP        | 573999     |
| CNN        | 834159     |
| ResNet     | 172239     |
| ViT        | 1201327    |

从参数量上看，`ViT` 的参数规模最大，`ResNet` 反而是四者中参数量较小的一类，但其测试表现最好。这说明在当前任务中，模型性能并不单纯由参数规模决定，更关键的是模型结构是否适合字符图像分类任务。

## 4. 结果分析

### 4.1 ResNet 相比 CNN 的优势

`CNN` 和 `ResNet` 都基于卷积操作，能够直接利用图像中的局部空间结构，例如边缘、拐点、闭环和笔画连接关系。这种局部归纳偏置对于字符识别非常有效，因为字符类别本身主要依赖局部笔画组合。

`ResNet` 在 `CNN` 的基础上引入了残差连接。残差连接的作用在于保留输入到输出的快捷路径，使网络在前向传播时能够更稳定地复用已有特征，在反向传播时也能够缓解梯度衰减问题。相比普通卷积网络，残差结构通常更容易训练，也更容易在更深的层次上学习到稳定而有效的判别特征。

本次实验中，`ResNet` 的准确率高于 `CNN`，说明残差结构确实提升了模型的特征提取与优化能力。即使 `ResNet` 的参数量并不大，它仍然取得了最好的综合结果，这反映出结构设计优于单纯堆叠参数。

### 4.2 ViT 与卷积模型的差异

`ViT` 将图像划分为多个 `patch`，再通过自注意力机制建模不同区域之间的关系。与卷积网络依赖固定大小卷积核不同，`ViT` 更擅长捕捉全局依赖关系，因此在理论上具有更强的全局建模能力。

但在本实验所使用的 `EMNIST Balanced` 数据集上，图像尺寸较小、任务目标明确，类别区分主要依赖字符的局部笔画结构。这种情况下，卷积模型天然具备更强的归纳偏置，因此往往更容易学习到有效特征。`ViT` 虽然表现优于 `CNN` 和 `MLP` 中的部分基线，但没有超过 `ResNet`，说明在中小规模字符识别任务上，卷积结构仍然具有明显优势。

### 4.3 MLP 表现较弱的原因

`MLP` 在输入阶段会直接将图像展平成一维向量，因此像素之间原本的二维空间关系不会被显式保留。模型虽然仍然可以从大量训练样本中学习到一定的判别模式，但相比卷积网络和 Transformer，它缺乏针对图像任务的结构先验。

因此，`MLP` 的性能通常作为图像分类任务中的基础对照模型较为合适。在本实验中，`MLP` 的结果最弱，也与这一结构特点一致。

## 5. 结论

综合四项指标与模型结构分析，可以得到以下结论：

- `ResNet` 在本实验中取得了最好的综合性能，说明残差结构能够有效提升卷积网络在字符识别任务中的表达能力与训练稳定性。
- `CNN` 具备较强的局部特征提取能力，整体表现稳定，但相比 `ResNet` 仍略有差距。
- `ViT` 具有较强的全局建模能力，但在小尺寸灰度字符数据集上，其结构优势没有完全发挥出来，整体表现略逊于 `ResNet`。
- `MLP` 缺乏图像空间结构建模能力，因此在四种模型中表现最弱。

总体来看，在 `EMNIST Balanced` 这类以局部笔画结构为主的字符分类任务中，带残差连接的卷积模型仍然是最合适的选择。
