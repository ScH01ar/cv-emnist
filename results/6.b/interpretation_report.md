# 模型可解释性分析报告

## 1. 分析目的

本部分通过可视化 `CNN`、`ResNet` 和 `ViT` 在测试样本上的关注区域，分析模型在进行字符分类时主要依赖了输入图像中的哪些区域，以及这些区域是如何影响最终预测结果的。

本次分析所选样本的真实标签为 `g`。为了便于对比，脚本自动选择了一个同时被 `CNN`、`ResNet` 和 `ViT` 正确分类的测试样本，使不同结构的模型能够围绕同一个字符进行解释分析。

## 2. 特征图与预测结果的关系

在 `CNN` 和 `ResNet` 中，卷积层会生成多个特征图。每个特征图都可以理解为某一种局部模式检测器在整张图像上的响应结果。

- 浅层特征图通常关注边缘、转折、局部纹理等低层视觉信息。
- 深层特征图则会逐渐组合出更接近语义的字符结构，例如闭环、长竖笔画、交叉结构、弯钩等。

模型最终并不是直接依据原始像素进行分类，而是依据这些深层特征图中哪些通道被激活、激活发生在什么位置来生成类别分数。也就是说，如果某些特征图在字符关键笔画位置上有较强响应，那么与该字符类别对应的输出分数就更可能被提高。

因此，特征图反映的是“模型检测到了什么模式”，而这些模式的空间位置和强度，会直接影响最终分类结果。

## 3. Grad-CAM 对 CNN / ResNet 的解释

为了进一步说明哪些区域真正推动了模型作出当前预测，本实验使用 `Grad-CAM` 对 `CNN` 和 `ResNet` 的最后一层卷积特征进行可视化。

`Grad-CAM` 的核心思想是：对目标类别分数相对于最后卷积层特征图求梯度，再用这些梯度作为通道权重，对特征图进行加权求和，最终得到一张热力图。热力图中响应更高的区域，表示这些区域对当前预测类别的贡献更大。

对于当前样本：

- `CNN` 预测结果为 `g`，置信度为 `0.987`。
- `ResNet` 预测结果为 `g`，置信度为 `0.978`。

如果热力图主要集中在字符主体笔画上，说明模型的判断依据是图像中真正具有判别性的部分，而不是背景噪声或无关区域。这通常意味着模型的决策更加可信。

从结构角度看，`ResNet` 与普通 `CNN` 的主要区别在于残差连接。残差连接能够保留前层信息并改善梯度传播，因此 `ResNet` 的特征表示通常更加稳定，关注区域也往往更完整、更贴近整个字符结构，而不仅仅局限于某个局部强响应区域。

## 4. 注意力权重对 ViT 预测结果的影响

`ViT` 的工作方式不同于卷积网络。它会先将输入图像切分为多个 `patch`，再把每个 `patch` 看作一个 token，通过自注意力机制学习不同区域之间的关系。

在本实验中，可视化的是最后一层中 `CLS token` 对各图像 `patch` 的注意力权重。其含义是：在生成最终分类表示时，模型更依赖哪些图像区域的信息。

对于当前样本：

- `ViT` 预测结果为 `g`，置信度为 `0.978`。

当某些 patch 的注意力权重较高时，说明这些区域在最终预测中起到了更重要的作用。如果高权重区域覆盖了字符的关键结构，例如闭环、竖线、交叉点或弯曲部分，就说明 `ViT` 在聚合全局信息时确实抓住了分类所需的核心区域。

与 `Grad-CAM` 不同，注意力权重热力图并不是直接表示“哪一块区域提升了类别分数”，而是表示“模型在整合信息时更重点参考了哪些区域”。因此，它更适合解释 `ViT` 是如何在全局范围内组织视觉信息并形成最终判断的。

## 5. 三种可视化结果的含义对比

- `Feature maps` 展示的是卷积通道对局部模式的响应情况，用来说明模型检测到了哪些视觉结构。
- `Grad-CAM` 展示的是对当前预测类别贡献最大的空间区域，用来说明模型为什么会把样本判为当前类别。
- `ViT attention map` 展示的是分类 token 在信息聚合过程中重点参考的 patch，用来说明模型是如何关注不同图像区域并形成整体判断的。

三者共同构成了从“局部特征检测”到“类别证据聚合”再到“全局关系建模”的解释链条。

## 6. 结论

通过对 `CNN`、`ResNet` 和 `ViT` 的可视化分析，可以得到以下结论：

- 卷积网络的特征图主要反映局部笔画和结构模式，说明其分类依据主要来自字符内部的局部空间特征。
- `Grad-CAM` 能进一步定位出对预测结果贡献最大的区域。如果热力图与字符主体高度重合，说明模型的决策基础较合理。
- `ResNet` 由于引入残差连接，通常能形成更稳定、更完整的字符级关注区域，这也解释了它在性能上通常优于普通 `CNN`。
- `ViT` 通过注意力机制在全局范围内整合不同 patch 的信息，其关注区域更具有全局性，能够体现出模型对字符整体结构关系的建模能力。

总体而言，无论是 `CNN / ResNet` 的特征图与 `Grad-CAM`，还是 `ViT` 的注意力热力图，本质上都在说明同一个问题：模型并不是平均地看待整张图像，而是会重点关注那些最能区分类别的关键区域，并基于这些区域作出最终预测。
